acme

发表于 2024-03-18 更新于 2024-03-20 阅读次数：
acme.sh是一个非常简单易用的ACME协议的客户端，可以用来申请Let's Encrypt的免费SSL证书。

在之前的帖子配置IIS实现反向代理中已经带到过acme.sh，这里详细介绍一下。

安装

无需root权限。

  1. 将acme.sh安装到~/.acme.sh，并创建一个shell的alias，alias acme.sh=~/.acme.sh/acme.sh，放进.bashrc。

  2. 创建cronjob，每天0点检测所有证书是否需要更新。

验证域名所有权，获取证书

验证域名有两种方式，http和dns。

http

--webroot指定网站根目录，-d指定要验证的域名。

dns

不建议使用dns验证，要配合dns api，比较麻烦。

安装证书

nginx有两处需要注意：

  1. ssl_certificate指向/etc/nginx/ssl/fullchain.cer，而非/etc/nginx/ssl/<domain>.cer。
  2. 使用service nginx force-reload而不是reload，因为reload不会重新加载ssl_certificate。
